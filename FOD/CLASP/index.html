Clasp introduction

Clasp is a open source instant messaging project with the intent of provided a free, secure and private alternative to the multi-national MSN Messenger and AIM networks.

Key features:
  • Encrypted data transfer.
  • Fast XML based messaging.
  • Server/Client available in many different langueages / platforms.
  • File Transfer Support.
  • You're choice on how you run your service
