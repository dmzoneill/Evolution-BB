Welcome to evoBB

What is evoBB?

evoBB is the hottest new forum software free for use on your website or online community. Unlike its' competitors which only support one type of database evoBB is revolutionary in the way that it supports many types of SQL based databases, and has done since 0.1.

How much is it?

It's open-source, meaning it's free.

What do I need to run it?

PHP3+
An SQL based database, yours is likely to be mySQL.
5MB server space (or less!)
5MB database space (or less!)

Where can I get it?
