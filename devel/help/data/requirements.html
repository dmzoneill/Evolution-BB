The EvoBB requirements are quite simple really!

Operating System - >

Well when it comes to web servers you cant beat linux, Windows (95,98,nt,me,2000,xp) just aint up there when it comes to stability and support.
I recommend linux but the board has been known to work nicely on Windows 98 ME and 2000 professional.

Web Server - >

Apache would be the ideal web server although in most cases any web server will be adequate.

PHP- >

PHP 3 and above is supported. PHP 4 is the is the current release so there should be no problems there.

Sql Database- >

Mysql -> Tested extensively and known to work flawlessly
Ibase -> Tested rarely and its stability is not known
Msql -> Tested rarely and its stability is not known
Mssql -> Tested rarely and its stability is not known
ifx -> Tested rarely and its stability is not known
Postgresql -> Tested rarely and its stability is not known
Sybase -> Tested rarely and its stability is not known
