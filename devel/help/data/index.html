Welcome to the long awaited release of EvoBB 0.4!

For those of you which have not downloaded this file from the official site,
EvoBB is a free bulletin board community management system, which supports any system running PHP and a sql database.
For further information read the Requirements page.

The EvoBB system is simple to set up and to ensure your success we compiled this guide to help you through the process.

All you need before starting the installation is a few handy tools to help get your Bulletin Board on the net.
Before You continue make sure you have a web space provider which conforms to our Requirements.
Well lets get onto these tools.


Needed + Handy Tools
FTP < File Transfer Protocol >
To get the files onto your web host you will need a ftp client below is a list of client that you can use depending on your Operating System Windows Users :
Cute Ftp available for donwload @ Global Scape software Developers.

Linux Users :
Gftp comes with most linux distros, if its not installed depending on your ditribution you can install an rpm off your distro cd which then will make it avalaible through KDE or Gnome menus.

Text Editor < For editing files >
Some time it is handy to have a text editor on hand to edit files if needed. Note : EvoBB comes with a program to do the editing for you.

Windows Users :
Notepad which is accessable from your start menu > programs > accessories > notepad will do fine

Linux Users :
Emacs or the VI editors are free are are usually distributed with most linux distros, if its not installed depending on your ditribution you can install an rpm off your distro cd which then will make it avalaible through KDE or Gnome menus.

PhpMyAdmin < For troubleshooting >
For trouble shooting database problems and learning more about sql this handy tool may be of some users. If your adventuroues :P.
PhpMyAdmin is available for donwload @ Source Forge OSDN network.

PhpTriad < For testing >
Php Triad is a packaged bundle of apache, php, mysql, perl and other tools to help people run web servers and scripting easilty on a windows machine.
Linux users have all if this on there distribution cd.
PHP Triad is available for download @ Cnet Downlaods.
